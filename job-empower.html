Associate Engineer Automation Quality

Company Name: Empower

Job Details

  • Job Type: Full Time
  • Eligible Branches: Bachelor’s degree in Computer Science, Information Technology
  • Batch: Fresher - 2Yrs Experience
  • Location: India
  • CTC: As per company standards

Role Explained (Simple)

Design and development of low-medium complexity automated test scripts. May assist more senior engineers with moderately complex automated test scripts to continue developing skills. Execute lower complexity automated test cases and report results. Identify, document and track, with guidance as needed, software defects to resolution using defect tracking tools like JIRA. Document test cases, test results, and defects clearly and concisely. Collaborate with the development team to understand application functionality and testing requirements. Maintain and update existing basic automated test scripts. Ensure that automated tests are up-to-date with the latest application changes and enhancements. Follow automation quality best practices. Required to do manual testing when needed Partner with more senior engineers to resolve issues and obtain guidance as needed.

Required Skills

  • Good understanding of software testing principles and methodologies.
  • Understanding of API & UI functional testing.
  • Familiarity with Java programming language (Core concepts) Understanding of SQL database testing is a plus. Good understanding of Test management tool ZE preferred. Understanding of version control systems (Git) is a plus. Willingness to learn and adapt to new technologies and methodologies is preferred.
  • Basic knowledge of automation testing tools (e.g., Selenium, Karate, JUnit, TestNG).
